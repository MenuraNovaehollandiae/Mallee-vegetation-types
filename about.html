Living Landscape mallee vegetation-type map

This repository contains an interactive vegetation-type map for the Living Landscape study region in the South Australian mallee. The map was developed using Sentinel-2 surface reflectance imagery, terrain variables, vegetation and moisture indices, field-verified training data, and Random Forest classification in Google Earth Engine.

Six mapped classes are shown: lunettes/playas, shrubby mallee, belah woodland, Triodia mallee, shrubland, and derived grasslands. The final classified raster was post-processed using a majority filter to reduce isolated pixel-level noise.

This vegetation map was developed as a regional decision-support layer and provides an input to related Living Landscape mapping products, including mapped State-and-Transition Model condition states and mallee bird habitat suitability predictions.

Raster values

Raster value Class
1 Lunettes / playas
2 Shrubby mallee
3 Belah woodland
4 Triodia mallee
5 Shrubland
6 Derived grasslands

This map should be interpreted alongside field knowledge, classification uncertainty, and the limitations of satellite-derived vegetation mapping.
