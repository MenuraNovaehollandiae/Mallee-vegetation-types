Living Landscape mallee vegetation-type map

This repository contains an interactive web map showing mapped vegetation types across the Living Landscape study region in the South Australian mallee.

The vegetation map was developed using Sentinel-2 satellite imagery, terrain variables, field-verified vegetation data, and Random Forest classification in Google Earth Engine. The resulting vegetation classes were used to support expert-elicited State-and-Transition Models and habitat suitability assessments for the mallee bird community.

Overview

The vegetation-type map classifies the Living Landscape study region into six mapped classes:

Raster value Class
1 Lunettes / playas
2 Shrubby mallee
3 Belah woodland
4 Triodia mallee
5 Shrubland
6 Derived grasslands

The final map is intended as a decision-support layer for visualising broad vegetation patterns across the Living Landscape study region. It also provides a key input to mapped State-and-Transition Model condition states and habitat suitability predictions.

Study region

The map covers the Living Landscape case study region in the South Australian mallee. The region of interest was delineated as the Living Landscape study area. Large ephemeral lakes and wetlands were masked from the analysis before classification.

How to use the map

Open the interactive vegetation-type map using the button above. The map can be used to explore broad spatial patterns in vegetation type across the Living Landscape study region.

It is intended as a regional decision-support layer and should be interpreted alongside field knowledge, classification uncertainty, and the limitations of satellite-derived vegetation mapping.

Relationship to other map products

This vegetation-type map provides an input layer for related Living Landscape map products, including mapped State-and-Transition Model condition states and mallee bird habitat suitability predictions.
